# Expense Tracker Web Application

A full-stack Expense Tracker Web Application built using the MERN stack (MongoDB, Express.js, React.js, Node.js) that helps users manage their personal finances efficiently by tracking income, expenses, balances, and financial trends through interactive dashboards and data visualizations.

---

## Live Features

This application allows users to:

- Securely register and log in using JWT Authentication
- Upload profile images during signup
- Add, view, and delete Income transactions
- Add, view, and delete Expense transactions
- Track Total Balance, Total Income, and Total Expenses
- View financial summaries using Pie Charts
- Analyse recent expenses using Bar Charts
- Analyse expense trends using Line Charts
- Download income and expense reports
- View recent transaction history
- Access a responsive and modern dashboard UI

## Authentication System
The application uses JWT authentication.

### Features:
- Protected routes
- Token-based login sessions
- Axios interceptors for automatic token handling
- Auto redirect to login when token expires
- User context for state management

## Challenges Solved 
Some major debugging and development challenges included:
- JWT authentication middleware issues
- Upload image export/ import errors
- Date filtering for 30-day and 60-day analytics
- React state refresh after add/delete operations
- Dynamic chart rendering issues in Recharts
- UserContext synchronisation problems
- Protected route hadnling after login
These were solved through backend debugging, frontend state optimisation and API repsonse handling improvements.
